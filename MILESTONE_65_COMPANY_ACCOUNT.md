تم إغلاق الشاشات 294–320 وربطها بطبقة API وقاعدة البيانات بدل أي بيانات تجريبية ثابتة.

## المنفذ

- الملف الشخصي وبيانات الشركة والمستندات المؤمنة والفروع والعناوين.
- دليل مستخدمي الشركة: إضافة وتعديل ونقل بين الفروع وتعطيل مع إلغاء الجلسات النشطة.
- دعوات بمدة 7 أيام ورمز عشوائي مخزن كـSHA-256، مع قبول الدعوة وإنشاء الحساب.
- أدوار مخصصة ومصفوفة صلاحيات، مع منع إسناد أدوار المنصة لحسابات العملاء.
- مستويات الموافقات ومسؤولوها وحدودهم المالية وSLA، وحد شراء منفصل لكل مستخدم.
- إدارة مراكز التكلفة وسجل تدقيق مؤسسي معزول حسب الشركة.
- الهوية البصرية وبيانات الفوترة والضرائب وشروط الدفع والعقود والتنبيه والتجديد.
