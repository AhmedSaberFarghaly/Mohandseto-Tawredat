# Demo Seed

بيانات العرض المترابطة تُنشأ تدريجيًا مع كل وحدة داخل `Infrastructure/*Seeder.cs`.

الكتالوج الحالي يُنشئ 12 قسمًا رئيسيًا، 40 قسمًا فرعيًا، 10 علامات، 6 وحدات بيع، و250 منتجًا مع الأسعار والكميات والخصائص والـvariants.

بيانات الشركات والطلبات والمخزون وRFQ ستُضاف مع المراحل المقابلة حتى تظل العلاقات صحيحة وقابلة للاختبار.
