## 1. وصف المشروع الحالي

منصة B2B لتوريد احتياجات الشركات (مصر) — تطبيق عميل موبايل + لوحة إدارة/CRM + API مركزي.
مصدر الحقيقة: ملف تصميم 759 صفحة (756 شاشة: 368 عميل + 388 إدارة + نظام تصميم) + وثيقة مواصفات تنفيذية شاملة.

## 2. التقنيات

| الطبقة | التقنية | الحالة |
|---|---|---|
| Backend | ASP.NET Core (.NET 10) + EF Core | يبني ويعمل، `/health` سليم |
| قاعدة البيانات | SQLite (تطوير) → SQL Server (إنتاج) | `InitialFoundation` + `CatalogExpansion` مطبقتان |
| لوحة الإدارة | Next.js 16 + TypeScript (App Router) | RTL + Dashboard + Products CRUD، يبني |
| تطبيق العميل | Flutter 3.32 (Android/iOS/Web) | Auth + Home + Catalog، analyze نظيف |
| CI | GitHub Actions (بناء الثلاثة) | مفعل على main/develop |
