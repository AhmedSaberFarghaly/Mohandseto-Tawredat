# Milestone 43% — Cart Operations

اكتملت الحالات التجارية للشاشات 109–126، وأصبحت السلة تحمي السعر والمخزون وتدعم دورة الشراء المتكررة للشركات.

## المنجز

- حفظ سلة كاملة باسم واستعادتها لاحقًا مع حفظ السلة الحالية تلقائيًا عند التعارض.
- كوبونات tenant-scoped بنسبة أو قيمة ثابتة، بحد أدنى وسقف وتاريخ وصلاحية استخدام.
- إعادة حساب الخصم والضريبة والإجمالي وتثبيت الكوبون داخل الطلب.
- التقاط سعر الصنف عند الإضافة، اكتشاف تغيره، ومنع Checkout حتى تأكيد السعر الجديد.
- اكتشاف انخفاض المخزون عن الكمية ومنع Checkout حتى التصحيح.
- ملاحظة مستقلة لكل صنف تنتقل إلى OrderItem، مع ملاحظة عامة للطلب.
- استمرار دعم السلة المختلطة للمنتجات العادية والمخصصة بسعر العرض النهائي.

## التحقق

- 26 اختبار Backend ناجحًا، منها كوبون وملاحظات وتغير سعر وحفظ/استعادة وتوفر جزئي.
- 9 اختبارات Flutter و`flutter analyze` بلا مشاكل.
- migration `CartOperations` مطبقة على قاعدة فارغة.
