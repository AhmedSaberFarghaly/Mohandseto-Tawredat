# الأصول البصرية المؤقتة

ملف التصميم يحتوي على صور منتجات داخل mockups وليست أصولًا تجارية أصلية قابلة للاستخراج والنشر منفردة بجودة مناسبة.

الحالة الحالية:

- صور الأقسام والعلامات والمنتجات مسجلة في قاعدة البيانات بمسارات `asset://` واضحة.
- تطبيق Flutter يعرض الصورة الحقيقية عند رفعها، ويرجع إلى Product Visual متسق فقط لمسارات Seed المؤقتة.
- لوحة الإدارة تدعم رفع JPG/PNG/WebP آمنًا وتحديد الصورة الرئيسية وحذفها، والـAPI يقدمها عبر URL قابل للتخزين المؤقت.
- لا توجد صور Base64 داخل قاعدة البيانات.
- لا توجد صور تحمل Watermarks.

قبل الإصدار التجاري النهائي، تُستبدل مسارات Seed من نوع `asset://` بصور مرخصة عبر مسار الرفع العامل حاليًا، ثم يُنقل storage provider من القرص المحلي إلى Object Storage مع نسخ متعددة المقاسات. غياب ملفات الصور التجارية الأصلية لا يوقف وظائف الرفع والعرض أو اختبارات البيانات.
